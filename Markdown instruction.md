## Списки

Чтобы добавить ненумерованные списки, необходимо пункты выделить звездочкой (*).
Пример:

* Элемент 1
* Элемент 2


Чтобы добавить нумерованные списки, необходимо пункты пронумеровать.
Пример:

1. Пункт один
2. Пункт два
